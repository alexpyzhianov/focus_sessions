# Focus session

A tool to help you focus

## How to use

1. Create data folder
```
mkdir $HOME/focus_sessions
```

2. Install dependencies
```
brew install ffmpeg
```

3. Make `focusSession.sh` available everywhere
```
ln -s /Users/you/repos/focusSession.sh /Users/you/bin/focus
```

4. Use it
```
focus 60
```
